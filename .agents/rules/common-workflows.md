Workflow for a userscript monorepo using `vite-plugin-monkey` and `typescript`:

- Full workflow (install, build, test): `pnpm run all`
- Install dependencies: `pnpm install`
- Run dev server (watch and serve userscript): `pnpm run dev`
- Build userscript for production: `pnpm run build`
